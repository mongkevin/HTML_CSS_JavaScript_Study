CSS 기본 선택자

선택자란?

특정 html 요소를 선택하고자 할때 사용하는 기능
해당 요소를 선택해서 원하는 스타일과 기능을 적용시킬수 있다.

1.모든(전체) 선택자(*)

모든 요소들을 다 선택하고자 할때 사용 

        *{
            스타일속성: 값;
            스타일속성: 값;
        }

2.태그 선택자(태그명)

해당 문서 내에 같은 태그들 모두 선택할때 사용 

        태그명{
            스타일속성: 값;
            스타일속성: 값;
        }

3.클래스 선택자(.클래스)

해당 문서 내에 같은 클래스명을 가진 태그들 선택할때 사용 

        .클래스명{
            스타일속성: 값;
        }
  • 클래스 선택자1
  • 클래스 선택자2
  • 클래스 선택자3
  • 클래스 선택자4
  • 클래스 선택자 없음

4.아이디 선택자(#아이디명)

해당 문서내에 해당 아이디값을 가진 태그를 선택할때 사용 단 해당 요소 id값은 고유하게 작성하여야한다 (고유하지 않으면 맨위에서 찾게된 아이디값을 가진 요소가 선택됨) 

        #아이디명{
            스타일속성: 값;
        }
