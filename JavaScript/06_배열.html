배열

자바스크립트에서는 변수 선언시 별도의 자료형 지정이 없기때문에
어떤 자료형의 값이던 하나의 배열 공간에 담을 수 있다(자바의 컬렉션 유사)

배열의 선언

배열의 선언시 배열의 크기를 지정한 채로 선언하거나, 크기를 지정하지 않아도 된다. 배열의 크기 제약이 없음

Array 객체의 메소드

1)indexof(찾고자하는 요소): 배열의 해당요소가 위치해있는 인덱스를 반환

concat(배열,배열,....): 여러개의 배열을 결합하고자할때 사용

3) reverse(): 배열에 담긴 값들을 역순으로 바꿔주는 메소드

4)sort(): 배열안에 담긴 값들을 오름차순으로 정렬시켜주는 메소드

5_1)push(추가할 요소): 배열의 맨뒤에 요소를 추가하고 배열의 크기를 반환된다. 5_2)pop(): 배열의 맨 뒤 요소를 제거하고 제거한 요소를 반환

6_1)unshift(추가요소): 배열의 맨앞에 요소 추가 후 배열의 크기 한환 6_2)shift(): 배열의 맨 앞에 요소 제거 후 제거 요소 반환

7_1)slice(시작, 끝): 배열 안의 요소들을 추출해주는 메소드 1_2)splice(시작, 제거수, 추가값): 배열의 요소를 추출해서 제거 및 추가, 추가값은 생략가능

8)join([구분자]) / toString(): 배열에 담긴 값들을 하나의 문자열로 합쳐 반환해주는 메소드
