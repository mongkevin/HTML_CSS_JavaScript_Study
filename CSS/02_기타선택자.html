1.속성 선택자

스타일을 부여하고자하는 요소 내에 작성되어 있는 속성을 이용하여 선택하는 방법 선택자 뒤에 []를 이용하여 속성과 속성값을 제시하여 선택하는 방법 

        선택자[속성=속성값] : 해당 속성값을 가질때
        선택자[속성~=속성값] : 특정 단어를 속성값으로 가질때(띄워쓰기 기준)
        선택자[속성|=속성값] : 특정 단어를 속성값으로 가질때(시작단어 유무)
        선택자[속성^=속성값] : 속성값이 특정단어로 시작할때
        선택자[속성&=속성값] : 속성값이 특정단어로 끝날때
        선택자[속성*=속성값] : 속성값이 특정 요소가 포함될때
                        {
                스타일 속성: 값;
        }

3.동위(같은 레벨) 선택자

동위 관계 (같은 례벨)에 있는 뒤에 위치한 요소를 선택할때 사용 

        *a요소 뒤에 있는 b요소 하나만 선택
        a+b{
            스타일 속성 : 값;
        }

        *요소 뒤에 있는 모든 b요소 선택
        a~b{
            스타일 속성 : 값;
        }

4.반응 선택자

사용자의 움직임에 따라 선택이되거나 안되는 선택자 

        *해당 요소에 클릭이 되었을 경우 스타일 부여하고자
        선택자: active{
            스타일 속성: 값;
        }
        *해당 요소에 마우스가 올라가는 순간 스타일 부여
        선택자: hover{
            스타일 속성: 값;
        }

5.상태 선택자

요소의 상태에 따라서 선택되는 선택자 

        1)checked(체크된) 상태의 요소
        선택자: checked{
            스타일 속성: 값;
        }
    

        2)초점(focus)이 맞춰진 input요소 선택
        선택자:focus{
            스타일속성: 값;
        }

        }

        4)비활성화 되어있는 요소들 선택
        선택자:disabled{
