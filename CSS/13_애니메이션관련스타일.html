애니메이션 관련 스타일

속성을 점차적으로 변화시켜 움직이는 효과를 내는 기법
트랜지션과 비교했을때 시작 스타일과 끝 스타일을 부드럽게 이어주는 기능을 한다는 것을
같지만 애니메이션은 중간에 원하는 위치에서 @keyframe을 이용하여 스타일 지정이 가능하다.



keyframes

애니메이션 시작지점과 끝지점의 스타일을 정하고 keyframes의 이름을 지정
시작 지점과 끝지점 두개만 설정할 경우 from to로 설정한다
중간에 작업이 있으면 %로 구분하여 설정한다. 

        @keyframes 키프레임명{
            선택자{
                스타일;
            }
        }
